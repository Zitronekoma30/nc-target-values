\abstract{
The aim of our project is to implement an alternative method of choosing target values for classification with neural networks.  Instead of using traditional one-hot-encoding, this method involves assigning custom "class-values" to each of the according classes. Additionally we aim to find non-class values which will replace the previous 0s across all classes. 

Our idea is to capture the models' natural predictions before (and during) training and use these to refine the class and non-class values as training progresses.

This approach requires a different rule for determining the predicted class: rather than choosing the maximal output value, we select the neuron whose output value is closest to the corresponding target value and therefore has the lowest error; a method we will refer to as the "minimum error rule".

The goal of our project is to increase training efficiency while maintaining acceptable output quality (i.e. classification accuracy).
To achieve this, we will attempt to further specify some of the methods we have been introduced to and evaluate the performance of these approaches on the MNIST dataset.
}
